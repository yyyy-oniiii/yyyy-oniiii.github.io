📗 Routine 2021_정보처리기사 필기 전체요약본

2021년 정보처리기사 필기시험(정기 2회) 합격을 위해 각 과목/챕터 별 중요 내욕을 요약해놓은 목록입니다.
모든 포스팅은 2021 시나공 정보처리기사 필기 교재의 기출 빈도와 중요도를 기준으로 작성되었습니다

2021년 3월부터 약 2달에 걸쳐 직접 공부한 내용을 실시간으로 작성했으며
교재만으로 부족한 내용들은 추가로 보강해놓았습니다.

특히 시험을 위한 기초적인 C언어와 IT 관련 배경지식이 부족하신 분들은 본 글의 최하단 참고자료 목록 의 보강자료들을 참고하시기 바랍니다.

🚩 2021 정보처리기사 시험일정/응시자격


🧙‍♂️ 2021 정보처리기사 1과목 | 소프트웨어 설계

🧙‍♂️ 1-1 요구사항 확인

  • 소프트웨어 생명주기 모델(+스크럼,XP 등)
  • UML 다이어그램
  • 요구사항 개발(정의/분석/확인)
  • 요구사항의 유형
  • 현행 시스템
  • 개발 기술 환경 정의


🧙‍♂️ 1-2 요구사항 확인

  • 사용자 인터페이스(UI)
  • 네비게이션
  • UI 설계도구
  • UI 요구사항
  • 품질 요구사항
  • UI 프로토타입
  • UI 설계서
  • UI 유용성 평가(상세설계, 감성공학)


🧙‍♂️ 1-3 어플리케이션 설계

  • 소프트웨어 아키텍처(모듈화/추상화/정보은닉)
  • 아키텍처 패턴
  • 객체지향(Object-Oriented) 기법의 특징
  • 캡슐화
  • 럼바우의 분석 기법
  • 객체지향 설계 원칙(SOLID)
  • 모듈의 품질(결합도&응집도)
  • 코드&디자인 패턴


🧙‍♂️ 1-4 인터페이스 설계

  • 시스템 인터페이스 요구사항
  • 송/수신 데이터 식별
  • 미들웨어
  • 소프트웨어의 분류
